Trinity

Trinity is a 2D plotting library integrated with AgentSmith written in javascript. It is easy to use, can work in web browsers and coordinate with AgentSmith and Neo. Moreover, Trinity designs stylish figures, which offers more choices to understand and apply various statistical analysis methods. At this point, we give priority to development of some frequently used visualizing tools for machine learning such as line plots, scatter plots, and contour plots.

Simple code

Usage of Trinity is similar to matplotlib (a major graph plotting library in python). You can draw graphs easily with simple codes. 

var plt = new Trinity('#sine-curves');	// Create Trinity instance attached to a DOM object
plt.plot(x, y, 'b-');			// Plot data points (x,y) on 2D space
plt.xlabel('x values');			// Set label for x axis
plt.ylabel('y values');			// Set label for y axis
plt.legend(['Some data']);		// Create legend
plt.show();				// Show graph
